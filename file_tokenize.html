file_tokenize¶

vsm.corpus.util.corpusbuilders.file_tokenize(text)¶

file_tokenize is a helper function for file_corpus.

Takes a string that is content in a file and returns words and corpus data.

Parameters:text (string) – Content in a plain text file.
Returns:words : List of words. Words in the text tokenized by :meth: word_tokenize. corpus_data : Dictionary with context type as keys and corresponding tokenizations as values. The tokenizations are np.arrays.
