filter_by_suffix¶

vsm.corpus.util.filter_by_suffix(l, ignore)¶

Returns elements in l that does not end with elements in ignore.

Parameters:
  • l (list) – List of strings to filter.
  • ignore (list) – List of suffix to be ignored or filtered out.
Returns:

List of elements in l whose suffix is not in ignore.

Examples

>>> l = ['a.txt', 'b.json', 'c.txt']
>>> ignore = ['.txt']
>>> filter_by_suffix(l, ignore)
['b.json']
