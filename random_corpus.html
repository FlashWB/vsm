random_corpus¶

vsm.corpus.util.corpusbuilders.random_corpus(corpus_len, n_words, min_token_len, max_token_len, context_type='context', metadata=False)¶

Generates a random integer corpus.

Parameters:
  • corpus_len (int) – Size of the Corpus.
  • n_words (int) – Number of words to draw random integers from.
  • min_token_len (int) – minimum token length used to create indices for corpus.
  • max_token_len (int) – maximum token length used to create indices for corpus.
  • context_type (string, optional) – A type of tokenization. Default is ‘context’.
  • metadata (boolean, optional) – If True generates metadata. If False the only metadata for the corpus is the index information.
Returns:

Corpus object with random integers as its entries.
