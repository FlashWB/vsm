word_tokenize¶

vsm.corpus.util.word_tokenize(text)¶

Takes a string and returns a list of strings. Intended use: the input string is English text and the output consists of the lower-case words in this text with numbers and punctuation, except for hyphens, removed.

The core work is done by NLTK’s Treebank Word Tokenizer.

Parameters:text (string) – Text to be tokeized.
Returns:tokens : list of strings
