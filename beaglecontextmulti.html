BEAGLE Context Model (Multiprocessing)¶

class vsm.model.beaglecontext.BeagleContextMulti(corpus, env_corpus, env_matrix, context_type='sentence')¶
Parameters:
  • corpus (Corpus object) –
  • env_corpus (Corpus object) –
  • env_matrix (2-D array) –
  • context_type (string, optional) – Name of tokenization stored in corpus whose tokens will be treated as documents. Default is sentence.
Attributes :
  • context_type (string)

    Name of tokenization where tokens will be treated as documents.

  • sents (list of arrays)

    Tokens by context_type retrieved from corpus.

  • dtype (np.dtype)

    Numpy dtype of env_matrix.

  • env_matrix (2-D array)

    Beagle environment matrix.
Methods :
  • train

    Takes an optional argument n_procs, number of processors, and trains the model on the number of processors. n_procs is 2 by default.
