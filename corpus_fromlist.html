corpus_fromlist¶

vsm.corpus.util.corpusbuilders.corpus_fromlist(ls, context_type='context')¶

Takes a list of lists or arrays containing strings or integers and returns a Corpus object. The label associated to a given context is context_type prepended to the context index.

Parameters:
  • ls (list) – List of lists or List of arrays containing strings or integers.
  • context_type (string, optional) – A type of tokenization.
Returns:

A Corpus object built from ls.
See Also:

vsm.corpus.Corpus

Examples

>>> ls = [['a', 'b'], ['c'], ['d', 'e']]
>>> c = corpus_fromlist(ls, context_type='sentence')
>>> c.view_contexts('sentence', as_strings=True)
[array(['a', 'b'], dtype='|S1'),
 array(['c'], dtype='|S1'),
 array(['d', 'e'], dtype='|S1')]
>>> c.context_data
[array([(2, 'sentence_0'), (3, 'sentence_1'), (5, 'sentence_2')], 
      dtype=[('idx', '<i8'), ('sentence_label', '|S10')])]
