Here are the following variations of Beagle Model.

The Beagle model was first featured in Representing Word Meaning and Order Information in a Composite Holographic Lexicon by Jones and Mewhort.

Beagle Composite - the Beagle Composite model combines Beagle Context and Beagle Order model in user-defined ratio.

Beagle Context model has 2 convenient classes for training.

Beagle Context - To get a better sense of a word, Beagle context model considers the co-occurring words.

Beagle Order model has 2 convenient classes for training.

Beagle Order - considers the order of a word in the sentence.

Bealge Environment - random environment vector model.

Here is a blog entry about `BEAGLE model<https://inpho.cogs.indiana.edu/datablog/info/info-beagle/>`_ in the InPho DataBlog.
