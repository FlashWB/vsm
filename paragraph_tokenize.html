paragraph_tokenize¶

vsm.corpus.util.paragraph_tokenize(text)¶

Takes a string and returns a list of strings. Intended use: the input string is English text and the output consists of the paragraphs in this text. It’s expected that the text marks paragraphs with two consecutive line breaks.
