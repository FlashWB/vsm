dir_tokenize¶

vsm.corpus.util.corpusbuilders.dir_tokenize(chunks, labels, chunk_name='article', paragraphs=True)¶

dir_tokenize is a helper function for dir_corpus().

Takes a list of files, chunks and labels for the chunks and returns words and corpus data.

Parameters:
  • chunks (list) – List of files. chunk_name indicates what a chunk represents.
  • labels (list) – List of filenames where files equal chunks.
  • chunk_name (string-like, optional) – The name of the tokenization corresponding to individual files. For example, if the files are pages of a book, one might set chunk_name to pages. Default is articles.
  • paragraphs (boolean, optional) – If True, a paragraph-level tokenization is included. Defaults to True.
Returns:

words : List of words. words in the chunks tokenized by :meth: word_tokenize. corpus_data : Dictionary with context type as keys and corresponding tokenizations as values. The tokenizations are np.arrays.
