sim_top_top¶

LDAGibbsViewer.sim_top_top(topic_or_topics, weights=None, print_len=10, filter_nan=True)¶

Takes a topic or list of topics (by integer index) and returns a list of topics sorted by the cosine values between a given topic and every topic.

Parameters:
  • topic_or_topics (string or list of strings) – Query topic(s) to which cosine values are calculated.
  • weights (list of floating point, optional) – Specify weights for each topic in topic_or_topics. Default uses equal weights (i.e. arithmetic mean)
  • print_len (int, optional) – Number of topics printed by pretty-pringing function Default is 10.
  • filter_nan (boolean, optional) – If True not a number entries are filtered. Default is True.
Returns:

a LabeledColumn object A 2-dim array containing topics and their cosine values to topic_or_topics.
