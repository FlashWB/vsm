file_corpus¶

vsm.corpus.util.corpusbuilders.file_corpus(filename, nltk_stop=True, stop_freq=1, add_stop=None)¶

file_corpus is a convenience function for generating Corpus objects from a a plain text corpus contained in a single string.

file_corpus will strip punctuation and arabic numerals outside the range 1-29. All letters are made lowercase.

Parameters:
  • filename – File name of the plain text file.
  • nltk_stop (boolean, optional) – If True then the corpus object is masked using the NLTK English stop words. Default is False.
  • stop_freq (int, optional) – The upper bound for a word to be masked on the basis of its collection frequency. Default is 1.
  • add_stop (array-like, optional) – A list of stop words. Default is None.
Returns:

c : a Corpus object Contains the tokenized corpus built from the input plain-text corpus. Document tokens are named documents.
See Also:

vsm.corpus.Corpus, file_tokenize(), vsm.corpus.util.apply_stoplist()
