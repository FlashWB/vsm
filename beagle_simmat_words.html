simmat_words¶

BeagleViewer.simmat_words(word_list, sim_fn=<function row_acos_mat at 0x4478848>)¶

Calculates the similarity matrix for a given list of words.

Parameters:
  • word_list (list) – A list of words whose similarity matrix is to be computed.
  • sim_fn (string, optional) – A similarity function from functions in vsm.linalg. Default is :meth: row_acos_mat.
Returns:

an IndexedSymmArray object n x n matrix containing floats where n is the number of words in word_list.
