simmat_topics¶

LDAGibbsViewer.simmat_topics(k_indices=[])¶

Calculates the similarity matrix for a given list of topics.

Parameters:k_indices (list, optional) – A list of topics whose similarity matrix is to be computed. Default is all topics in the model.
Returns:an IndexedSymmArray object n x n matrix containing floats where n is the number of topics considered.
See Also:vsm.viewer.similarity.simmat_topics()
